HTML5 New Form Attributes

HTML5 introduce mew type of form input tags. Some of these new features are datalist, output, keygen and etc.

HTML CODES

How to create a Datalist


[input list="browsers" name="browser"] [datalist id="browsers"] [option value="Internet Explorer"]Internet Explorer[/option] [/datalist]

How to create Required field

[input type="text" name="usrname" required]

How to create a Placeholder

[input type="text" name="fname" placeholder="First name"]

How to check if email is valid

[input type="email" name="email" required placeholder="Enter a valid email address"]
